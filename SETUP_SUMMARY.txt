### 3. ✅ Cải thiện Chat Interface

### 4. ✅ Cải thiện Context Message
Khi người dùng gửi câu hỏi, thêm context tự động:
```
"Tôi là một người [giới tính]. Thông tin chi tiết về tôi:
- Tuổi: X tuổi
- Chiều cao: Xcm
- Cân nặng: Xkg
- Chỉ số BMI: X

Dựa trên thông tin này, vui lòng cung cấp tư vấn..."
```

### 5. ✅ Memory-only Storage
- Chat conversation lưu trong React state (memory)
- Khi người dùng thoát → tất cả dữ liệu bị xóa
- Khi làm mới trang → reset conversation
- Không lưu vào database

### 6. ✅ Luồng hoạt động
```
1. Người dùng điền: Giới tính, Tuổi, Chiều cao, Cân nặng
2. Nhấn "Bắt đầu chat với AI"
3. Chuyển sang giao diện chat
4. AI greeting hiển thị tự động với thông tin chi tiết
5. Người dùng chat tự do, AI trả lời dựa vào context
6. Nhấn "Kết thúc chat" → quay về form, xóa tất cả history
```
